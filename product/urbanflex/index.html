Introducing ADDEVA

ADDEVA –「常に付加価値のある商品を作り続ける」

ADDEVA(アデバ)の名前の由来は、「Added Value（付加価値）」という言葉にあります。私たちは、単なる商品提供に留まらず、日常をより豊かにする製品を創り出すことを社名に誓いました。

10年以上にわたりベトナムで日本向けの鞄づくりに携わってきた経験を活かし、日本人管理のもと現地で生産・検品を実施。ベトナム直送に近い体制で中間コストを抑え、日本品質、高機能なビジネスバッグ・ビジネスリュックを中心に販売しています。

Features

3WAYビジネスリュック「UrbanFlex」の共通特徴（2R-18 / 3REX-40）

“Ultra Light Design”

思わず声が出る軽さを実現。


軽量かつ防水性に優れた高品質素材を贅沢に使用し、これだけの多機能性とサイズを有しながら、2R-18(2room)でわずか1.1kg（付属品除く）という驚きの軽量化を実現しました。手に取った瞬間、「軽い！？」と思わず声に出してしまうほどの軽さ。ビジネスリュックとして毎日の通勤や出張での負担を大幅に軽減し、長時間の移動も快適にサポートします。

“AquaShield” 防水軽量ビジネスリュック


毎日の使用を想定し、防水性と軽量性を高次元で両立したビジネスリュックです。止水ジッパーを贅沢に使用し、単に防水素材を使用するだけではなく、何度も降雨テストを繰り返しながら、設計・縫製まで徹底的に見直しました。突然の雨にも中身を守り、ビジネスバッグとしてもストレスフリーな移動を実現します。

※浸水への防水性はありません。リュック使用時のみ想定。

“Flexible room”

荷物量に応じて形が変わる自在な収納空間。


「いつも同じ場所に収納して、迷わず取り出したい。」
2ルーム以上でポケットが豊富なリュックを選べば解決すると思われがちですが、実際に使ってみると意外にうまくいかないものです。ある部分はパンパンに膨らみ、別の部分はスカスカになって、バッグ全体が型崩れを起こしてしまいます。
そんな悩みを解決するのが、独自の「フレキシブルルーム」。仕切り板にマチを設けることで、収納物の量に応じて柔軟に形を変え、常に美しいシルエットを保ちます。荷物が少ない日でも多い日でも、ビジネスバッグとして型崩れすることなく快適にお使いいただけます。

“Smart Pockets”
時短を叶える収納設計

よく使う物に“最短動線”でアクセスできるよう、UrbanFlexは細部まで工夫した収納を随所に配置。 電車内・移動中・出張時など、ビジネスシーンごとに最適な取り出しやすさを追求したビジネスリュックです。


  1. ショルダーストラップポケット：スマホとカードサイズのポケットを装備。詳細はこちら
  2. 隠しポケット：PCルーム中央に、財布やパスポートケースが入る隠しポケットを装備。横持ちでもリュック使用でも、スムーズに出し入れ出来ます。
  3. 水筒（ペットボトル）ポケット：水筒や濡れた折りたたみ傘の収納に。詳細はこちら
  4. フロント上部ポケット：自転車や車のキーなど、毎日使う小物を整理して保管できます。
  5. フロント下部ポケット：内部にスマートフォン用ポケットを2つ装備。斜め配置で、横持ち時でも取り出しやすい工夫を施しています。
  6. フロント大ポケット：雑誌やノートが入る大型サイズ。毎日使う鍵を取り付けられるストラップフックと専用ポケットを装備します。

※モデルにより一部仕様が異なる場合があります。

“Smart Expandable Zip”
スマートエキスパンダブルジッパー

※この機能は 3REX-40 のみ搭載です。


ADDEVAの拡張ジッパーは、収納室を増やすのではなく、メイン収納室そのものを広げる設計です。これにより、幅のあるアイテムもスムーズに収納できます。さらに、「Wスライダー」を採用することで、下部のみをスマートに拡張可能。お弁当箱や少し大きめのアイテムを収納したい時に、必要な分だけ容量をアップできる便利な機能で、ビジネスバッグ兼トラベルリュックとして活躍します。

“Quick Fix Strap”

訪問先の直前まで、リュックスタイルで移動。


従来の3wayバッグは、ブリーフケースモードにする際、ショルダーストラップをジッパー内に完全収納する必要があり、手間と時間がかかっていました。ADDEVAは、ストラップを収納するのではなく、本体に固定する方式を採用。これにより、わずか2アクションで収納できるので、訪問先の直前までリュックとして快適に移動し、到着と同時にビジネスバッグ（ブリーフケース）スタイルへ瞬時に変身できます。

“3D Ergo Back Panel”
疲れを軽減させる背面設計


ショルダーストラップは幅広8cmで、クッション性の高い3Dメッシュを採用。肩の角度にフィットし、肩への負担を軽減します。背面は全面に3Dメッシュを採用し、十字の通気ルートを確保することで背中の蒸れを効果的に低減。ビジネスリュックとして、人間工学に基づいた設計により、長時間の使用でも快適な背負い心地を実現します。

“Wet Safe Pocket”
濡れた傘も入れられるペットボトルポケット


3wayビジネスリュックやビジネスバッグでペットボトルポケットを備えたものは多くありません。しかし、ADDEVA UrbanFlexはリュックとしての使用時間が長い方のために設計しているため、専用のペットボトルポケットを装備しました。ポケット底部にメッシュ素材を採用し、水分が下に流れる構造になっているので、ペットボトルはもちろん、濡れた折りたたみ傘を収納する際にも重宝します。

“Smart Access Strap”
ショルダーストラップポケット


リュックのショルダーストラップに、スマホと交通カードサイズのポケットを搭載。混雑した電車内でスマホが収納できず、「ここにポケットがあったらいいのに」と思ったことはありませんか？ADDEVAは、まさにそこにポケットを作りました。開発では、外観と機能の両立にポイントを置き、部分的に伸縮素材を使用することで、ポケットが目立たない設計になっています。ポケットのない服装で出かけた日、歩きながら連絡を待っているときなどに重宝します。携帯電話を湾曲から守る構造になっているので、安心して収納できます。

※カードポケットに、スキミング防止機能はございません。スキミング防止カードケースのご利用を推奨しています。

“Immigration Access Pocket”
空港利用を想定したスマート収納


ADDEVA UrbanFlexは、出張や旅行でも快適に使えるよう設計された3WAYビジネスリュックです。空港での保安検査や搭乗手続きでは、長蛇の列に並ぶこともあり、機内持ち込みスーツケースにオンキャリーして使用したい方も多いのではないでしょうか。ADDEVA UrbanFlexは、オンキャリーして使用する際に最も便利な位置にポケットを配置しました。また、リュックを前に抱えて持つときにもスムーズにアクセスできます。ポケットはパスポートケースがすっぽり入るサイズで、普段使いでも、マスクやモバイルバッテリー、イヤホンなどの小物を整理して保管できるよう設計しています。

“Rain Lock cable hole”
防水ケーブルホール


PCルーム内のモバイルバッテリーから、スマートにケーブルを引き出すことができます。ケーブルは、ショルダーストラップ上の専用ガイドを通して、ショルダーストラップポケットの携帯電話に接続可能。ケーブルホールは、独自の防水設計により、雨水の浸入をシャットアウト。従来のUSB接続ポートとは異なり、物理的な接続部がないため、長期使用による接触不良のリスクを根本から排除しています。

“Garment Guard”
シワを防止する衣類収納スペース


ビジネスルームの書類ポケットは、ジャケット等の衣類をシワを付けずに保管できる多機能ポケットです。衣類収納としてご使用される際は、両サイドのスライダーを開けてから収納し、スライダーを閉めて衣類をホールドして使用します。出張用ビジネスバッグとしても活用しやすい設計です。

3WAYビジネスリュック UrbanFlexについてのFAQ

2R-18 と 3REX-40 共通／相違点、ビジネスバッグとしての使い勝手に関するご質問

3REX-40は「拡張マチ」と「独立PCルーム」を備えた3WAYビジネスリュックです。
・ジッパーが少なく軽快さ重視のビジネスリュック → 2R-18
・PCや周辺機器を日常的に持ち歩く／荷物が多くなりがち → 3REX-40
・普段は少なめだが時々大荷物 → 軽快な2R-18をビジネスバッグ兼トラベルリュックとして使うのがおすすめです。

家庭用シャワーで背負い状態・約1分の社内テストでは、外装に軽度の染み込みはあるものの、内部への浸水はほぼありませんでした。
ただし完全防水ではありません（豪雨・長時間の降雨・水没・強い水圧は非対応）。
また外部ポケットはリュック使用時に雨が入りにくい向きで設計しており、ブリーフケース持ち（ビジネスバッグスタイル）では開口部から侵入する可能性があります。

PCスリーブは独立しています。対応目安は以下のとおりです。
・2R-18： 17インチ薄型まで対応。ビジネスリュックとして通勤用PCを想定しています。
・3REX-40： 17インチ“ゲーミング”クラスにも対応可能な余裕があります。
いずれも機種により外寸が異なるため、端末の外寸（幅・高さ・厚み）をご確認ください。

多くの航空会社で非拡張状態の機内持込を想定しています。拡張時は規定を超える場合があります。ご搭乗の航空会社のサイズ・重量規定を必ずご確認ください。

大型スマホまでの収納を想定しています。機種ごとの適合は端末の外寸とケース有無により異なるため、寸法ガイドは製品ページで順次更新します。

側面のWet Safe Pocketに収納できます。底部メッシュにより水が下に抜け、他の荷物への影響を抑えます。
目安として500mlペットボトルやスリム型水筒、濡れた折りたたみ傘に対応します。

2サイズともサコッシュ（Detach Pocket）が標準付属です。パスポート・スマホ・財布など軽量物の携行に適します（重量物の収納はお控えください）。

実は、初期サンプルでは、他社と同様に背面下部に配置していました。
しかし実際に財布などを収納してみたところ、PCや荷物を多めに入れた際に腰へ当たり痛みを感じることがあったため、現在の位置に変更しています。この場所の隠しポケットは、3WAYリュックだからこそ実現できる配置です。

3REX-40は20〜40Lへ可変する想定です（荷物の形状・パッキングにより体感は変わります）。

拡張マチは約8cm広がります。
Wスライダーの特性を活かし、下側のみを開ければ、弁当箱や厚みのある小物だけをスマートに収容できます。上部の見た目をすっきり保ちつつ、必要分だけ容量を追加できるのがメリットです。

背面スリットでキャリーハンドルに固定できます。走行時の横ブレを抑えます。重心が上がると不安定になるため、重い荷物はキャリー側に寄せると安定します。

汚れは中性洗剤を薄めた布でやさしく拭き取り、ぬるま湯で固く絞った布で拭き取り、陰干ししてください。中性洗剤以外は絶対に使用しないでください（洗濯機・乾燥機・漂白剤・溶剤・アイロン不可）。

素材はオックスフォードポリエステル＋PUコーティングです（細部仕様はモデルにより異なる場合があります）。付属品はサコッシュ標準付属。
重量は付属品を除く実測値です。その他の最新仕様は各商品ページでご確認ください。

返品・交換・修理はご購入先（各モール）のポリシーに準拠します。まずは各モールの商品ページのガイドをご確認ください。お困りの場合はinfo@addeva.jpへご連絡ください。

いずれも各モール（販売チャネル）により異なります。最新の価格・在庫・出荷〜到着目安・送料・配送会社は、ご購入予定のモールの商品ページでご確認ください。

撥水が弱くなっても内部への防水性（浸水しにくさ）は変わりません。どうしても撥水性を復活させたい場合は、素材適合の市販撥水剤を目立たない箇所で試験したうえで、説明書に従って薄く均一に施工してください（推奨ではありません／自己責任にてご対応ください）。

ジッパー内に完全収納する必要はなく、本体側の固定ポイントへ2アクションで留めて固定します（バックル固定はありません）。
これにより移動直前までリュックとして使え、到着時に素早くビジネスバッグ（ブリーフ持ち）へ切り替え可能。固定後はぶらつきを抑える設計です。

USB外付けポートは搭載していません。
その代わりに防水ケーブルホール（Rain Lock）を採用。PCルームのモバイルバッテリーからケーブルを通し、ショルダーストラップのガイド経由でスマホへ給電できます。物理ポートがないため、接触不良や水侵入のリスクを抑えられます。

目安はシャツ1〜2枚または薄手ジャケット1枚です。
使い方は、両サイドのスライダーを開放→衣類を平らに入れる→スライダーを閉じてホールド。たたみジワを伸ばしてから収納すると、シワをより抑えられます。
